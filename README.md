# GramaUrja ⚡🌿

## Smart Village Power Monitoring System

GramaUrja is an Android application developed to modernize rural electricity monitoring and outage reporting through real-time digital infrastructure. The app enables citizens to report electricity-related issues instantly while helping administrators monitor, track, and manage power failures efficiently.

This project was developed as part of an internship project using modern Android development technologies including Jetpack Compose, Firebase, Firestore, Google Maps, and real-time data synchronization.

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/GramaUrja.git
```

---

## 2️⃣ Open in Android Studio

Open:

```text
Android Studio → Open Project → GramaUrja
```

---

## 3️⃣ Configure Firebase

### Enable:

* Firebase Authentication
* Cloud Firestore
* Google Sign-In
* Google Maps SDK

---

## 4️⃣ Add google-services.json

Place the file inside:

```text
app/google-services.json
```

---

## 5️⃣ Run Project

Click:

```text
Run App ▶️
```

# 📊 Project Workflow

1. User logs into application
2. User reports power issue
3. GPS coordinates are captured
4. Data stored in Firestore
5. Admin panel receives realtime updates
6. Google Maps displays outage markers
7. Monitoring dashboard updates instantly
